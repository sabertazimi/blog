# Fancy Code Block

## Markup Code Block

```html
<h2>JSX Heading 2</h2>

<abbr title="HyperText Markup Language">HTML</abbr> is a lovely language.

<section>And here is *markdown* in **JSX**!</section>
```

```xml
<div># this is not a heading but *this* is emphasis</div>

<div>This is a `p`.</div>
```

## TypeScript Code Block

```ts
const foo = function (bar) {
  return bar++;
};

foo(5);

some.code();
```

```tsx
{
  (function () {
    const guess = 0.44;

    if (guess > 0.66) {
      return <span style={{ color: 'tomato' }}>Look at us.</span>;
    }

    if (guess > 0.33) {
      return <span style={{ color: 'violet' }}>Who would have guessed?!</span>;
    }

    return <span style={{ color: 'goldenrod' }}>Not me.</span>;
  })();
}
```

## Meta Code Block

Disable line number `noline`:

```ts noline
console.log('No line number!');
```

Disable copy button `nocopy`:

```ts nocopy
console.log('No copy button!');
```

Show code title `title="Awesome Code"`:

```ts noline nocopy title="Awesome Code"
console.log('Awesome!');
```

Live code block `live`:

```ts live title="Live Code"
console.log('Live!');
```
